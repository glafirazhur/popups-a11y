Структура модального окна

Что насчёт <dialog>?

Структура модального окна (HTML)

  
  <div class="modal">
    <div class="modal__header">
      <h2>Заголовок модального окна</h2>
    </div>
    <div class="modal__content">
      <!-- контент -->
    </div>
    <div class="modal__footer">
      <!-- кнопки -->
    </div>
    <button>X</button>
  </div>

Accessibility

Кнопка закрытия (сейчас)

  
    <button>X</button>

Прячем иконку от скринридера

  
    <button>
      <span aria-hidden="true">X</span>
    </button>

Добавляем label для кнопки

  
    <button>
      <span aria-hidden="true">X</span>
      <span class="visually-hidden">Закрыть окно</span>
    </button>

  </div>

Фокус - на кнопку закрытия окна

Зацикливаем фокус в окнe (focus trap)

Закрываем модальное окно

Возвращаем фокус на открывший элемент

Прячем модальное окно

  
    .hidden {
      display: none;
    }

Мы не должны никак попасть внутрь модального окна.
Можно протестировать с помощью Tab и скринридера
