HTML and a11y check-list

  1. Follow the right structure
  2. Give the role="dialog" for the modal
  3. Give the id="ID" for the heading
  4. Connect the heading to the modal: aria-labelledby="ID"
  5. Set the type for the window aria-modal="true"
